# django-discord-rolemanager


## Dev

### Prepare docker compose: 

```
version: '2.4'
services:
  app:
    build:
      context: ""
      dockerfile: Dockerfile
    volumes:
      - ./pogobackend:/usr/src/app
    restart: always
    environment:
      - "DEVELOPMENT=True"
    depends_on:
      - db
    ports:
      - 8000:8000

  # PostgreSQL database for Django
  db:
    image: postgres
    volumes:
      - postgres_data:/var/lib/postgresql/data
    environment:
      - POSTGRES_USER=postgres
      - POSTGRES_PASSWORD=postgres
      - POSTGRES_DB=postgres
    restart: always
    ports:
      - 5432

  # Redis for Celery
  redis:
    image: "redis:alpine"
  
  rabbitmq:
    image: rabbitmq:latest

networks:
  proxy:
    external: true

volumes:
  postgres_data:
```

### start
```
docker-compose up -d
```

```
docker-compose exec -it app sh
migrate.py createsuperuser 
```

1. go to localhost:8000/admin
1. login with your admin account
1. create discord app: https://discord.com/developers/applications   - redirect url to http: http://localhost:8000/accounts/discord/login/callback/
1. create a social application: 
   - provider: Discord
   - name: discord
   - clientid: from developer portal
   - secret: from developer portal
   - add new site localhost:8000
